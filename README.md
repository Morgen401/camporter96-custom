# Installation

* Exit Steam

* Download the aio package: https://github.com/babgozd/camporter96-custom/releases/download/v1.0/grid.zip

* Unzip contents of grid.zip to Steam\userdata\\[steamIDhere]\config\grid

* Enjoy!
